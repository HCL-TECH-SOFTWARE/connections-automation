# Component Pack Cache Service Redis → Valkey Migration (HCL Connections 8.0 CR13+)

Starting with HCL Connections 8.0 CR13, Valkey replaces Redis in the Component Pack.

**The scripts in this redis/ folder are deprecated. Use the unified scripts in `../cache-service/` instead.

The new scripts support both configuration types. Future releases may remove these legacy Redis scripts entirely.
